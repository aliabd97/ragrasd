أنت محلل نصوص متخصص في الكتب الدينية والتراثية. مهمتك: استخراج جميع المصادر والمراجع من نص كتاب "الشافي في الإمامة" للشريف المرتضى.

# المهمة

استخرج **كل** المصادر المذكورة في النص أدناه بدقة كاملة.

# أنواع المصادر المتوقعة

1. **مصادر رقمية مباشرة:**
   - `(1)` أو `[1]` أو `{1}`
   - مثال: "كما ذكر في الكتاب (12) أن..."

2. **مصادر نصية:**
   - "كما في كتاب الغارات..."
   - "روى الطبري في تاريخه..."
   - "ذكر ابن قتيبة في المعارف..."

3. **مصادر بصيغ مختلفة:**
   - قال في كتاب...
   - روى عن...
   - أخرج البخاري...
   - رواه مسلم...

# صيغة JSON المطلوبة

```json
{{
  "volume_number": {volume},
  "total_citations": X,
  "citation_formats_found": ["رقمية", "نصية مباشرة", "إسنادية"],
  "citations": [
    {{
      "citation_id": "cite_v{volume}_001",
      "appearance_in_text": "(1)" أو "كتاب الغارات",
      "source": {{
        "book_name": "اسم الكتاب الكامل",
        "author": "اسم المؤلف إن وُجد",
        "full_reference": "المرجع الكامل كما ورد في النص",
        "type": "primary" أو "secondary"
      }},
      "context": "الجملة أو الفقرة التي ورد فيها المصدر",
      "page_or_section": "رقم الصفحة إن وُجد",
      "reliability": "high" أو "medium" أو "low"
    }}
  ]
}}
```

# ملاحظات مهمة

1. **الدقة مطلوبة 100%:**
   - لا تفوّت أي مصدر مهما كان صغيراً
   - انتبه للمصادر المكررة - سجلها مرة واحدة مع ذكر تكرارها

2. **السياق:**
   - اذكر الجملة الكاملة التي ورد فيها المصدر
   - هذا مهم جداً للربط لاحقاً

3. **التصنيف:**
   - `primary`: مصدر أصلي (قرآن، حديث، كتب تاريخية قديمة)
   - `secondary`: مصدر ثانوي (شروح، تفاسير، دراسات)

4. **الترقيم:**
   - استخدم `cite_v{volume}_` متبوعاً برقم تسلسلي من 001
   - مثال: `cite_v1_001`, `cite_v1_002`, ...

# النص المراد تحليله

**الجزء:** {volume}

```
{full_text}
```

# التعليمات النهائية

- أخرج **فقط** JSON صحيح بدون أي نص إضافي
- تأكد من صحة الأقواس والفواصل
- استخدم escape للنصوص العربية بشكل صحيح
- إذا لم تجد مصادر في قسم معين، لا بأس - لكن تأكد من البحث جيداً

**ابدأ الآن!**
