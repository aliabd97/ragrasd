أنت محلل نصوص ديني متخصص في تقسيم الكتب التراثية إلى أقسام منطقية.

# المهمة

قسّم نص كتاب "الشافي في الإمامة" (الجزء {volume}) إلى **أقسام منطقية** مع ربط كل قسم بمصادره.

# خريطة المصادر المستخرجة مسبقاً

**إجمالي المصادر:** {total_citations}

**عينة من المصادر:**
```json
{citations_map}
```

# معايير التقسيم

1. **الموضوعية:**
   - كل قسم يتناول موضوعاً واحداً متكاملاً
   - الحجم المثالي: 500-2000 كلمة

2. **الحفاظ على المصادر:**
   - **حرج جداً:** كل مصدر يجب أن يُربط بقسمه
   - استخدم `appearance_in_text` للتعرف على المصدر في النص
   - ضع المصادر في القسم الذي ذُكرت فيه

3. **العناوين:**
   - عنوان واضح يعكس محتوى القسم
   - يفضل استخراجه من النص نفسه

# صيغة JSON المطلوبة

```json
{{
  "volume_number": {volume},
  "total_sections": X,
  "sections": [
    {{
      "section_id": "shafi_v{volume}_s001",
      "title": "عنوان القسم الواضح",
      "order": 1,
      "word_count": 1234,
      "text": "النص الكامل للقسم...",
      "citations": [
        {{
          "citation_id": "cite_v{volume}_001",
          "appearance": "(1)" أو "كتاب الغارات",
          "position_in_section": 523,
          "source": {{
            "book_name": "اسم الكتاب",
            "full_reference": "المرجع الكامل"
          }}
        }}
      ],
      "topics": ["الموضوع 1", "الموضوع 2"],
      "parent_doc": "shafi_v{volume}"
    }}
  ],
  "validation": {{
    "total_words_in_volume": X,
    "total_words_in_sections": Y,
    "citations_in_volume": {total_citations},
    "citations_in_sections": Z,
    "status": "complete" أو "warning: X citations missing"
  }}
}}
```

# ملاحظات حرجة

1. **عدم فقدان المصادر:**
   - قارن `citations_in_volume` مع `citations_in_sections`
   - يجب أن تتطابق الأرقام 100%
   - إذا اختلفت، راجع وأصلح

2. **عدم فقدان النص:**
   - مجموع كلمات الأقسام = كلمات الجزء الأصلي

3. **الترقيم:**
   - `shafi_v{volume}_s001`, `s002`, ...
   - استخدم 3 أرقام

4. **position_in_section:**
   - موقع ظهور المصدر داخل نص القسم (بالأحرف)
   - يساعد في الربط الدقيق لاحقاً
